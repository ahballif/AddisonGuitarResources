Scale Degrees Quiz

Here is a quiz that can help you practice memorizing the notes in each key. It is useful to know how each note relates to the root of a key. In the game below, you will be prompted with a key and a scale degree. Type the note name of the scale degree in that key.

By default the game only uses scale degrees of the major scale. To include all scale degrees, click the checkmark below.
