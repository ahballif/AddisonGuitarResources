Drones

Here are some resources for creating drones to practice with. Practicing scales with drones can be a useful exercise. Drones can help you to internalize the root of the key. With the root in mind, pay attention to how the other notes in the key sound in their relationship to the root.

First I have included a javascript tool that can play any drone in a few different octaves. After the javascript synthesizer tool I have included several links to some higher quality drones that can be found on YouTube.
